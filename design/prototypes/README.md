# 智能分账系统原型

## 项目概述

本项目是基于PRD和用户故事地图创建的智能分账系统高保真HTML原型。原型使用HTML、Tailwind CSS和FontAwesome实现，旨在为开发团队提供视觉和交互参考。

## 技术栈

- HTML5
- Tailwind CSS
- FontAwesome 图标库
- 原生JavaScript

## 项目结构

```
design/prototypes/
├── css/
│   ├── tailwind.css
│   └── styles.css (自定义样式)
├── js/
│   └── main.js (交互功能)
├── index.html (入口页面)
├── account-management.html (账户管理界面)
└── README.md (本文档)
```

## 已完成界面

1. **入口页面 (index.html)**
   - 所有界面的导航入口
   - 界面预览区域

2. **账户管理界面 (account-management.html)**
   - 多层级账户结构管理
   - 账户创建、编辑、删除功能
   - 账户详情查看

## 设计风格

- **视觉风格**：现代、简洁、专业的Web应用界面
- **色彩方案**：以蓝色为主色调，辅以绿色、黄色等功能色
- **响应式设计**：适配桌面和平板设备
- **暗黑模式**：支持明亮和暗黑两种主题

## 交互功能

- 侧边栏导航切换
- 下拉菜单
- 标签页切换
- 模态框
- 表格排序和筛选
